============================================
  Gemini Desktop - インストール手順
============================================

1. 「Gemini Desktop.app」を Applications フォルダにドラッグ

2. ターミナルで以下のコマンドを実行（初回のみ）:

   xattr -cr "/Applications/Gemini Desktop.app"

   ※ コード署名なしのため macOS Gatekeeper の
     解除が必要です

3. アプリを起動

============================================
  Gemini Desktop - Installation
============================================

1. Drag "Gemini Desktop.app" to Applications

2. Run in Terminal (first time only):

   xattr -cr "/Applications/Gemini Desktop.app"

   * Required to bypass macOS Gatekeeper
     for unsigned apps

3. Launch the app
